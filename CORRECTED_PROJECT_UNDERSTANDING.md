## Revised Crisis Detection

**Simple 3-Step Approach:**

**Step 1: Detect Extreme Distress**
```
Keywords: "want to die", "kill myself", "end it all", "suicide", etc.
Sentiment: Extremely negative + hopelessness
```

**Step 2: Provide Supportive Response**
```
"I can hear that you're going through something really difficult right now. 
I'm an AI companion and I'm not equipped to provide crisis support, but I 
want to make sure you have access to people who can help.

If you're in immediate danger, please call:
- 988 Suicide & Crisis Lifeline (US)
- 911 for emergencies
- [Local crisis resources]

I'm here to listen when you're ready to talk about other things."
```

**Step 3: Log for Platform Safety**
```
{
  "type": "safety_alert",
  "severity": "high",
  "user_id": "...",
  "timestamp": "...",
  "keywords": ["..."],
  "context": "User expressed extreme distress",
  "action_taken": "Provided crisis resources"
}
```

**That's it. No escalation, no professional intervention, no HIPAA.**
